---
title: ToggleButtons
---

A horizontal set of toggle buttons.

`youtube: https://youtu.be/kVEguaQWGAY`

## Example

```dart liveslice=ToggleButtons
ToggleButtons(
  children: <Widget>[
    Icon(Icons.ac_unit),
    Icon(Icons.call),
    Icon(Icons.cake),
  ],
  onPressed: (int index) {
    setState(() {
      isSelected[index] = !isSelected[index];
    });
  },
  isSelected: isSelected,
),
```
