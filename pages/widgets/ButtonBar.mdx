title: ButtonBar

An end-aligned row of buttons.

## Example

```dart liveslice=ButtonBar
ButtonBar(
  children: <Widget>[
    FlatButton(
      child: Text('Ok'),
      color: Colors.blue,
      onPressed: () {/** */},
    ),
    FlatButton(
      child: Text('Cancel'),
      color: Colors.blue,
      onPressed: () {/** */},
    ),
  ],
)
```
