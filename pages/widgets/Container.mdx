title: Container

A convenience widget that combines common painting, positioning, and sizing widgets.

`youtube: https://youtu.be/c1xLMaTUWCY`

## Example

```dart liveslice=Container
Center(
  child: Container(
    margin: const EdgeInsets.all(10.0),
    color: Colors.amber[600],
    width: 48.0,
    height: 48.0,
  ),
)
```
