[Flutter Widget Livebook](https://flutter-widget.live) یک پروژه تحت وب ایجاد شده به وسیله فلاتر برای آشنایی و کار با ویجت ها به صورت آنلاین می باشد.

افزونه کروم در دسترس می باشد. از مثال های آنلاین استفاده کنیید.

[![](/images/chrome-web-store.svg)](https://chrome.google.com/webstore/detail/flutter-widget-livebook/lnabimpogllgckbeoneoegflahpefomf)

## نمونه کد ها

### کارت

```dart liveslice=Card
// See: /widgets/Card
```

### AnimatedBuilder

```dart liveslice=AnimatedBuilder height=400px
// See: /widgets/AnimatedBuilder
```

### AnimatedPhysicalModel

```dart liveslice=AnimatedPhysicalModel
// See: /widgets/AnimatedPhysicalModel
```

### NestedScrollView

```dart liveslice=NestedScrollView mockup=iPhone7
// See: /widgets/NestedScrollView
```

## پاسخ و پرسش

اگر پیشنهادی یا سوالی در مورد این پروژه دارید ، میتوانید در گروه تلگرام با دیگران در میان بگذارید [Telegram Group](https://t.me/flutterwidgetlivebook).
