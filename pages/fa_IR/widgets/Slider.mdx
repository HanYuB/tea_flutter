A Material Design slider.

`youtube: https://youtu.be/ufb4gIPDmEs`

## Example

```dart liveslice=Slider
double _progress = 0;

@override
Widget build(BuildContext context) {
  return Slider(
    value: _progress,
    label: '$_progress',
    min: 0.0,
    max: 100.0,
    onChanged: (value){
      setState(() {
        _progress = value.roundToDouble();
      });
    },
  );
}
```
