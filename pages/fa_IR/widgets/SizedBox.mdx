A box with a specified size.

`youtube: https://youtu.be/EHPu_DzRfqA`

## Example

```dart liveslice=SizedBox
SizedBox(
  width: 200.0,
  height: 300.0,
  child: const Card(child: Text('Hello World!')),
)
```
