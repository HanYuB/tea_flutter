---
title: CupertinoSwitch
---

An iOS-style switch.

## Example

```dart liveslice=CupertinoSwitch
MergeSemantics(
  child: ListTile(
    title: Text('Lights'),
    trailing: CupertinoSwitch(
      value: _lights,
      onChanged: (bool value) { setState(() { _lights = value; }); },
    ),
    onTap: () { setState(() { _lights = !_lights; }); },
  ),
)
```
