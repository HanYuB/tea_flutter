title: OutlineButton

Similar to a FlatButton with a thin grey rounded rectangle border.

## Example

```dart liveslice=OutlineButton
OutlineButton(
  onPressed: () {
    /*...*/
  },
  child: Text(
    "Outline Button",
  ),
)
```
