title: Scrollbar

A material design scrollbar.

## 示例

```dart liveslice=Scrollbar mockup=iPhone7
Scrollbar(
  child: ListView.builder(
    itemCount: 100,
    itemBuilder: (context, index) {
      return ListTile(
        title: Text('Item $index'),
      );
    },
  ),
),
```
