title: Checkbox
---

A material design checkbox.

## 示例

```dart liveslice=Checkbox
bool _value = true;

Checkbox(
  value: _value,
  onChanged: (bool newValue) {
    setState(() {
      _value = newValue;
    });
  },
)
```
