---
title: PageView
---

A scrollable list that works page by page.

## 示例

```dart liveslice=PageView mockup=iPhone7
PageView(
  children: <Widget>[
    Container(
      color: Colors.pink,
      child: Center(
        child: Text('Page 1'),
      ),
    ),
    Container(
      color: Colors.cyan,
      child: Center(
        child: Text('Page 2'),
      ),
    ),
    Container(
      color: Colors.deepPurple,
      child: Center(
        child: Text('Page 3'),
      ),
    ),
  ],
)
```
