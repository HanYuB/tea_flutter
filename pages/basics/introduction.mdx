[Flutter Widget Livebook](https://flutter-widget.live) is a website built with Flutter for web to live preview widget samples online.

> Now we created a Chrome extension. Let your flutter docs support live samples.

[![](/images/chrome-web-store.svg)](https://chrome.google.com/webstore/detail/flutter-widget-livebook/lnabimpogllgckbeoneoegflahpefomf)

## Examples

### Card

```dart liveslice=Card
// See: /widgets/Card
```

### AnimatedBuilder

```dart liveslice=AnimatedBuilder height=400px
// See: /widgets/AnimatedBuilder
```

### AnimatedPhysicalModel

```dart liveslice=AnimatedPhysicalModel
// See: /widgets/AnimatedPhysicalModel
```

### NestedScrollView

```dart liveslice=NestedScrollView mockup=iPhone7
// See: /widgets/NestedScrollView
```
